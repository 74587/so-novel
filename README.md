# so-novel

![logo.ico](assets%2Flogo.ico)

## 介绍

交互式书源下载器（命令行操作）

可根据书名、作者搜索并下载小说

支持导出格式：epub、txt、html

结合以下阅读器使用更佳
- 电脑：[koodo-reader](https://www.koodoreader.com/zh)
- 手机：微信读书

提示：Apple Books（图书）不能章节自动定位，如果看到 500 章了，就需要从上一直向下拉，十分不便

## 使用

下载最新免安装版 https://github.com/pcdd-group/so-novel/releases

解压即用
